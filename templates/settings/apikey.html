Cómo usarla

Enviar un mensaje

curl -X POST {{ request.base_url }}api/v1/messages/send \
  -H "X-API-Key: {{ current_user.api_key }}" \
  -H "Content-Type: application/json" \
  -d '{
    "account_id": 1,
    "to": "573001234567",
    "type": "text",
    "content": { "text": "Hola desde Metabridge" }
  }'

  -H "X-API-Key: {{ current_user.api_key }}"
