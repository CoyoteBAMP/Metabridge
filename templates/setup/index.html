Guía de configuración

Sigue estos pasos para recibir mensajes reales en tu inbox.

1

Exponer tu servidor local a internet

Meta necesita llegar a tu servidor para enviarte los mensajes. En local, usa ngrok para crear un túnel público.

# Instala ngrok desde ngrok.com, luego ejecuta:

ngrok http 8000

Ngrok te dará una URL pública temporal.

2

Configurar el webhook en Meta for Developers

Ve a developers.facebook.com → tu App → Messenger o WhatsApp → Configuración de webhooks.

TU_URL_NGROK/api/webhook/meta
{{ meta_verify_token }}
{% for event in ['messages', 'messaging_postbacks', 'message_deliveries', 'message_reads'] %} {{ event }} {% endfor %}
3

Verificar que el webhook responde

Meta enviará un GET de verificación. Si muestra ✅, todo está bien.

  • Asegúrate de que ngrok sigue corriendo
  • Verifica que el Verify Token sea: {{ meta_verify_token }}
  • La URL debe terminar en /api/webhook/meta
4

Probar con un mensaje real

  1. Abre Messenger desde cualquier cuenta de Facebook
  2. Escribe a la página de Facebook que conectaste
  3. El mensaje debe aparecer en tu Inbox en segundos
  4. Responde desde el inbox — llegará al usuario en Messenger
